# 3D 포트폴리오 갤러리 - 기획서

## 📋 프로젝트 개요

**컨셉**: 3D 갤러리를 걸어다니며 프로젝트를 전시 형태로 보여주는 인터랙티브 포트폴리오

**핵심 기능**
- 1인칭 시점 이동 (WASD + 마우스)
- 프로젝트 전시 패널 (이미지 + 정보)
- 명화 상호작용 (F키 → 진동 애니메이션)
- 보물찾기 퀘스트 (보물 수집 → 명화 특별 설명 해금)

---

## 🎮 주요 기능

### 1. 플레이어 이동
- WASD + 마우스 카메라 회전
- CharacterController 사용

### 2. 전시 패널
- 프로젝트 이미지 + 제목
- E키 상호작용 → 상세 정보 UI

### 3. 명화 상호작용
- F키 → 진동 애니메이션 (0.5초, ±3도)
- PlayerDetector (3m 감지)
- 효과음 + 힌트 UI

### 4. 보물찾기 퀘스트
- 갤러리 곳곳에 보물 3-5개 숨김 (발광 효과)
- F키로 수집 → 인벤토리 저장
- 특정 보물 수집 시 → 연결된 명화의 특별 설명 해금
- 진행도 UI: "보물: 2/5"



## 🔧 구현 순서

1. **기본 환경** (10분): 씬, 바닥, 조명
2. **플레이어** (30분): 이동 + 카메라
3. **전시 패널** (30분): 프리팹 + 상호작용
4. **UI** (1시간): 정보 패널 + 조작법
5. **명화 시스템** (45분)
   - PaintingInteractable + PlayerDetector
   - Unity AI 애니메이션: "Create painting shake animation, ±3 degrees Z-axis rotation, 0.5 seconds, 12-15 rapid shakes with bounce easing"
6. **보물찾기** (1시간)
   - TreasureData (ScriptableObject)
   - TreasureItem (수집)
   - QuestManager (진행 관리)
   - PaintingNPC (보물 연동)
7. **레이아웃** (30분): 배치 + 테스트

## 🎯 게임 플로우

1. 갤러리 입장 → "보물: 0/5" UI 표시
2. 탐험 중 반짝이는 보물 발견 → F키 수집
3. 명화 접근 → F키 상호작용
   - **보물 미수집**: 기본 대화 + 진동 애니메이션
   - **보물 수집 완료**: 특별 설명 해금 (프로젝트 비하인드 스토리)
4. 모든 보물 수집 (5/5) → 히든 엔딩
