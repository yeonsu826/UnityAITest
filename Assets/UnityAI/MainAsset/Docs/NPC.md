# NPC
NPC는 미술관에서 전시된 미술품을 설명해주는 NPC이다.

## NPC에 대한 상호작용
1. 플레이어가 NPC에 인터렉션을 해서 설명글이 팝업
   1) F키를 눌러서 인터렉션을 해야함.
   2) 인터렉션 했을 때 간단한 버튼 효과음이 나와야 함.
2. F키를 눌러서 인터렉션을 해야함.
3. 인터렉션 했을 때 간단한 버튼 효과음이 나와야 함.
4. 설명글이 UI형태로 나와야 한다.
5. 플레이어가 NPC한테 일정거리 떨어지면 자동으로 설명글 UI 가 사라진다.
